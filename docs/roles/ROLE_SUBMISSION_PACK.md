This pack is prepared to satisfy the instructor requirements:

- Design of role-specific modules with clear interfaces
- Implementation of modules applying security principles
- Functional testing per role
- Role-specific documentation with design, implementation, and testing

2. `FRONTEND_ROLE.md`
3. `BACKEND_ROLE.md`
